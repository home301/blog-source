# About

AI 도구 활용, 로컬 GPU 환경 구축, Linux 시스템 관리 등 직접 부딪히며 얻은 실전 경험을 기록한다. 검색해도 안 나오는 삽질의 해법, 공식 문서가 알려주지 않는 현실적인 팁을 다룬다.

## 주요 주제

- **AI/개발** — Stable Diffusion, ComfyUI, LLM 등 AI 도구의 로컬 설치와 활용
- **Linux/시스템** — Pop!_OS, GPU 드라이버, PyTorch 환경 구축
- **게임 개발** — 2D/Live2D, 인디 게임
- **생활리뷰** — 제품 리뷰, 맛집, 여행

## home301이라는 이름

home301은 "홈 서버 301번지"에서 출발했다. 개인 서버에서 시작한 프로젝트들을 기록하는 공간이라는 의미를 담고 있다.
